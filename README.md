# Nova Tech Website

A responsive corporate website for **Nova Tech**, a company offering innovative technology solutions. The website is built using **HTML, Bootstrap 5, and custom CSS**, designed to provide a modern, user-friendly experience on both desktop and mobile devices.

## Features

- **Responsive Design:** Optimized for desktop, tablet, and mobile screens.  
- **Sticky Navbar:** Elegant navigation with a mobile-friendly hamburger menu.  
- **Hero Section:** Eye-catching introduction with headline and call-to-action.  
- **Services Page:** Detailed list of company services with CTA buttons.  
- **About Page:** Company story, mission, vision, and goals.  
- **Contact Page:** Contact form and embedded Google map.  
- **Recent Projects Section:** Showcases company projects with links.  
- **Newsletter Signup:** Collects user emails for updates.  
- **Client Testimonials:** Highlighting client satisfaction.  

## Technologies Used

- HTML5  
- Bootstrap 5  
- CSS3
